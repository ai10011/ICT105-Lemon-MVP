# Prototype Folder

This folder contains the working prototype for **Lemon - Smart Campus Lost-and-Found System** for Lab 10.

The interactive web prototype is located in the `project/` directory. It demonstrates full lost-and-found reporting, item search/filtering, detail viewing, administration status management, and analytics using HTML, CSS, JavaScript, and `localStorage`.

Files included in the web prototype (`prototype/project/`):
- `index.html` - Homepage and search landing screen
- `form.html` - Lost and found item report form
- `records.html` - Item catalog with search and category filters
- `detail.html` - Item detail view and ownership claim verification
- `admin.html` - Admin panel for staff status updates
- `dashboard.html` - Analytics summary dashboard
- `login.html` & `signup.html` - User and admin access screens
- `css/styles.css` - Custom UI styling and layout rules
- `js/` - Client-side interaction logic and data handling
- `server.js` / `server.py` - Local dev server scripts
