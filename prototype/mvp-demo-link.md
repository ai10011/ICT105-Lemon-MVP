## Testing Notes & Instructions

To test the prototype on your own machine, please follow these steps:

1. **Launch the App**: Open the `prototype/project` folder and double-click the `index.html` file to open it in your preferred web browser, or run `node server.js` inside `prototype/project`.
2. **Interact Freely**: This is an interactive web prototype used to demonstrate the UI and user flow. You can click through the screens to see how reporting, searching, and admin functions work.
3. **Roles to Explore**:
   - **Student (User)**: Start at the homepage (`prototype/project/index.html`) to report a lost/found item (`form.html`) or search for items (`records.html`).
   - **Campus Staff (Admin)**: Navigate to `prototype/project/admin.html` or `dashboard.html` to view inventory metrics and update item statuses.
