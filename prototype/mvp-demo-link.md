## Testing Notes & Instructions

To test the prototype on your own machine, please follow these steps:

1. **Launch the App**: Open the `prototype` folder and double-click the `index.html` file to open it in your preferred web browser.
2. **Interact Freely**: This is a static frontend prototype used to demonstrate the UI and user flow. You can click through the screens to see how reporting, searching, and admin functions are intended to look and feel.
3. **Roles to Explore**:
   - **Student (User)**: Start at the homepage (`index.html`) to report a lost item or search for a found one.
   - **Campus Staff (Admin)**: Navigate to the `dashboard.html` or `admin.html` pages to view inventory metrics and update item statuses.
