# Problem Notes

## Selected Case

## Target Respondents

## Observed / Reported Problem

Many respondents reported that finding lost items is difficult because there is no single platform for reporting and searching. Information is often shared through different channels, making it easy for reports to be missed. Users spend considerable time searching for lost items and are often unsure where to report or check for updates.

## Notes for MVP Direction
What should the team consider when designing the MVP later?

- Create a single centralized platform for reporting and searching lost items.
- Allow users to submit both lost-item and found-item reports.
- Include search and filtering by category, date, and location.
- Support photo uploads to improve item identification.
- Display item status (Lost, Found, Claimed, Returned).
- Provide notifications when a possible match is found.
- Keep the reporting process simple and fast to encourage participation.
- Prioritize commonly lost items such as student IDs, access cards, USB drives, calculators, and wallets.
- Ensure the system is easy to use on both desktop and mobile devices.
- Design the platform to reduce manual work for security and administrative staff.
