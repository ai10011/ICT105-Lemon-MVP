# User Consent Statement

This prototype has been developed for educational purposes as part of the **ICT105 Fundamental Technology Entrepreneurship** course project for **Lemon - Smart Campus Lost-and-Found System**. You are invited to participate in a usability test by completing several tasks and providing feedback on your experience.

Participation is entirely voluntary. You may stop the test at any time without any penalty. Please do **not** provide any sensitive personal information (such as passwords, national ID numbers, or financial details). All information collected will be used solely for improving the prototype and completing the course project.

---

## Information Collected

The following information may be collected during the usability test:

- **Name, nickname, or anonymous ID** (optional)
- **Usability feedback score**
- **Comments on usability and user experience**
- **Task completion results and reporting workflow observations**

---

## Use of Data

The collected information will be used only for:

- Analyzing usability test results and task completion metrics.
- Improving the prototype design, search filters, and reporting functionality.
- Supporting the final ICT105 project report and presentation.

All responses will be treated confidentially and used for academic purposes only.
