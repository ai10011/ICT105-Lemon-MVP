# Lemon: Campus Lost & Found System

Lemon is a centralized, user-friendly platform designed to streamline the recovery of personal property on campus.

## Wireframe Specification

| Wireframe File | Purpose | Minimum Elements to Show | Related Requirements |
|----------------|---------|--------------------------|----------------------|
| `/wireframes/homepage.png` | Landing page / Entry point | Project title, search bar, main action buttons, navigation | FR-01, FR-02 |
| `/wireframes/input-form.png` | Report an Item | Form fields, image upload, privacy disclaimer, submit button | FR-03, FR-10, FR-11 |
| `/wireframes/record-list.png` | Browse Items | Filters, card list of items, **View Details** action | FR-05, FR-06, FR-08 |
| `/wireframes/detail-view.png` | Item Detail View | Item image, full description, location, claim action | FR-07, FR-08 |
| `/wireframes/dashboard.png` | User Dashboard | Profile summary, personal stats, pending matches, reports | FR-12 |
| `/wireframes/admin-view.png` | Admin Dashboard | Overview metrics, pending claims, recent reports list | FR-09, FR-08 |

## Wireframe Quality Rules

- Use consistent navigation and labels across all screens.
- Avoid decorative-only screens.
- Show realistic sample data instead of empty placeholders.
- Ensure every screen supports the final prototype requirements.
